RaspberryPI折腾Archlinux

发表于 2016-12-07 | 分类于 Linux

本文说明了RaspberryPI安装Archlinux并安装无线网卡驱动，成为路由的过程。

硬件

我的硬件是RaspberryPI一代B版本，不同的硬件应该到网站下载相对应的版本。

系统安装

安装过程可以见这里：

Replace sdX in the following instructions with the device name for the SD card as it appears on your computer.

Start fdisk to partition the SD card:

fdisk /dev/sdX

At the fdisk prompt, delete old partitions and create a new one:
Type o. This will clear out any partitions on the drive.
Type p to list partitions. There should be no partitions left.
Type n, then p for primary, 1 for the first partition on the drive, press ENTER to accept the default first sector, then type +100M for the last sector.
Type t, then c to set the first partition to type W95 FAT32 (LBA).
Type n, then p for primary, 2 for the second partition on the drive, and then press ENTER twice to accept the default first and last sector.
Write the partition table and exit by typing w.

Insert the SD card into the Raspberry Pi, connect ethernet, and apply 5V power.
Use the serial console or SSH to the IP address given to the board by your router.
Login as the default user alarm with the password alarm.
The default root password is root.

关于分区：

可能有的人跟我一样使用虚拟机而没有在电脑上安装linux系统而导致无法使用fdisk，在分区的时候为了使用虚拟机进行分区可以采用如下的方法（此处仅介绍了vmware中的方法，virtualbox应该也类似）：

  1. 插入SD卡
  2. 在虚拟机中添加实体硬盘（此处选择vmware推荐的SCSI）
  3. 然后就可以顺利的进行上述的安装操作了

安装完成后RaspberryPI会使用自动获取IP的方式，可以进行扫描得到IP然后进行登陆。ssh登陆只能使用alarm用户登录，不能使用root登陆。登陆系统以后可以使用su root切换为root用户。

Raspberry Pi超频

vim /boot/config.txt

armfreq：ARM处理器的频率，单位是MHz。默认值是 700。 最简单的方法是将“armfreq”的值改成“900”并加上“sdram_freq=500”。 一步秒杀，十分轻松。
