LAMP环境与typecho的安装

发表于 2015-12-21 | 分类于 Linux

LAMP环境

安装软件包

$ aptitude install apache2 mysql-server php5 php5-mysql

添加apache网站设置

新建/etc/apache2/sites-available/testsite.com文件，并添加如下内容：

<VirtualHost *:80>
    ServerAdmin admin@admin.com
    ServerName foolengineer.com
    ServerAlias blog.foolengineer.com www.foolengineer.com

    DocumentRoot /var/www/foolengineer.com
    <Directory />
            Options FollowSymLinks
            AllowOverride None
    </Directory>
    <Directory /var/www/foolengineer.com/>
            Options Indexes FollowSymLinks MultiViews
            AllowOverride None
            Order allow,deny
            allow from all
    </Directory>

    ScriptAlias /cgi-bin/ /usr/lib/cgi-bin/
    <Directory "/usr/lib/cgi-bin">
            AllowOverride None
            Options +ExecCGI -MultiViews +SymLinksIfOwnerMatch
            Order allow,deny
            Allow from all
    </Directory>

    ErrorLog ${APACHE_LOG_DIR}/error.log

    # Possible values include: debug, info, notice, warn, error, crit,
    # alert, emerg.
    LogLevel warn

    CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

并启用网站配置：

$ a2ensite testsite.com
$ service apache2 reload

mysql设置

mysql初始化安全设置：

$ mysql_secure_installation

新建数据库和用户并赋予用户权限：

$ mysql -u root -p
> CREATE DATABASE TestDatabase;
> GRANT ALL PRIVILEGES ON TestDatabase.* TO 'newuser'@'localhost' IDENTIFIED BY 'newuserpassword';
> FLUSH PRIVILEGES;

typecho的安装

安装依赖的软件包：

$ aptitude install php5-curl

然后按照官方的安装过程安装就ok了

# LAMP # typecho
