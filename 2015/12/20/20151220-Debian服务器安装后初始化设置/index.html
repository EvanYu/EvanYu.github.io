Debian服务器安装后初始化设置

发表于 2015-12-20 | 分类于 Linux

编辑~/.bashrc文件

根据需要修改~/.bashrc文件中的内容适合自己的使用。

修改源

修改为普通国内源请在网络上查询，使用DVD光盘源见Debian中使用DVD作为本地源。

安装NTFS支持

$ aptitude install ntfs-3g

sudoer设置

安装

$ aptitude install sudo

将普通用户加入sudo组（假设用户名为user1）

$ usermod -g sudo user1

基本编译环境的安装

aptitude install -y binutils  build-essential cmake gawk bison flex texinfo automake libtool cvs libncurses5-dev libglib2.0-dev gettext intltool subversion  git-all python-dev libssl-dev autoconf

ssh设置

  • 备份/etc/ssh/sshd_config文件
  • 修改如下项目
    • 端口：Port 2222
    • 只允许root采用密匙登录：PermitRootLogin without-password
    • 关闭X11转发：X11Forwarding no
    • 只允许采用密匙登录：PasswordAuthentication no

vim安装与设置

见编译包含python的vim。

安装缺失的驱动

  • 无线网卡驱动的编译参考BCM4311、Edup_ES-MS1578。
  • 关于笔记本的SD卡驱动：正常情况下笔记本的SD卡驱动不需要单独去安装，可以直接挂载/dev/sdb
