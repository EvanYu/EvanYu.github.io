Debian中使用DVD作为本地源

发表于 2015-12-20 | 分类于 Linux

在网络速度不佳或者没有网络的情况下，使用DVD作为本地源是一个不错的选择。

准备

  • 完整的Debian镜像（ISO）
  • 已安装完成的Debian系统

涉及文件

  • /etc/init.d/dvd-source
  • /etc/apt/sources.list

步骤

建立文件夹

mkdir /media/iso1
mkdir /media/iso2
mkdir /media/iso3

添加脚本/etc/init.d/dvd-source（根据自身ISO路径和名称修改）

#! /bin/sh
### BEGIN INIT INFO
# Provides:          EvanYu
# Required-Start:    $local_fs
# Required-Stop:
# Default-Start:     2 3 4 5
# Default-Stop:      0 1 6
# Description:       Mount the local dvd as package source.
### END INIT INFO

case "$1" in
    start)
        echo
        echo "Mounting Debian iso..."
        echo
        if [ ! -d "/media/iso1" ];then
            mkdir /media/iso1
            mkdir /media/iso2
            mkdir /media/iso3
        fi
    if [ ! -e "/var/run/dvd-source.run" ];then
        mount -o loop /home/debian_iso/debian-7.9.0-i386-DVD-1.iso /media/iso1/ >> /var/log/dvd_source.log 2>&1
        mount -o loop /home/debian_iso/debian-7.9.0-i386-DVD-2.iso /media/iso2/ >> /var/log/dvd_source.log 2>&1
        mount -o loop /home/debian_iso/debian-7.9.0-i386-DVD-3.iso /media/iso3/ >> /var/log/dvd_source.log 2>&1
        touch /var/run/dvd-source.run
    fi
        ;;

    stop)
        echo
        echo "Umounting Debian iso..."
        echo
        if [ -e "/var/run/dvd-source.run" ];then
            umount /media/iso1
            umount /media/iso2
            umount /media/iso3
            rm /var/run/dvd-source.run
        fi
        ;;

    *)
        echo
        echo "Usage: /etc/init.d/dvd-source start|stop"
        echo
        exit 1
        ;;
esac

exit 0

修改/etc/apt/sources.list为如下：

deb file:///media/iso1/ wheezy contrib main
deb file:///media/iso2/ wheezy contrib main
deb file:///media/iso3/ wheezy contrib main

如果是Debian8则将wheezy替换为jessie

启动并添加脚本到开机启动

$ service dvd-source start
$ update-rc.d dvd-source defaults

更新源

$ sudo aptitude update
# 源 # Debian # DVD
